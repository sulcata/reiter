Module: reiter/reject

Methods

(generator, inner) reject(predicate, iterable) → {*}

Skips all values in `iterable` for which predicate returns a truthy value. `predicate` is invoked with one argument: (value).
Parameters:
Name Type Description
predicate function Invoked per value.
iterable ForOfIterable The iterable.
Since:
  • 0.0.1
Source:
  • reject.js, line 6
See:
  • filter
Yields:
The next value not passing `predicate`.
Type
*
Example
reiter.reject(n => n % 2, [1, 2, 3, 4])
// => 2, 4
